# 数据科学导论大作业

## 文件夹结构

```
期末大作业/
├── main.ipynb                       # 主要代码文件（Jupyter Notebook）
├── 数据科学导论大作业报告.pdf           # 大作业报告文档
├── 数据科学导论期末汇报.pptx           # 期末汇报PPT
├── html/                            # HTML可视化文件目录
│   ├── flow_prediction_map.html     # 流量预测地图可视化
│   └── start&end_points.html        # 起点和终点可视化
└── model/                           # 模型文件目录
    └── flow_model.pth               # 训练好的流量预测模型
```

## 文件说明

- **main.ipynb**: 包含所有代码实现，包括数据处理、模型训练和结果分析等。
- **数据科学导论大作业报告.pdf**: 完整的大作业报告文档。
- **数据科学导论期末汇报.pptx**: 期末汇报使用的PPT。
- **html/**: 存放交互式可视化HTML文件，可在浏览器中打开查看结果。
- **model/**: 存放训练好的模型文件，用于预测和推理。
